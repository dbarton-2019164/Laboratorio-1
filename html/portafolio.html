Tecnologías que conozco

1.HTML

¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es un estándar fundamental en desarrollo web. Utilizado para estructurar y presentar contenido en páginas web. Consiste en etiquetas que definen elementos como encabezados, párrafos y enlaces. Permite incorporar multimedia y enlazar a otras páginas, facilitando la creación de contenido interactivo y accesible en la web.

2.Java

¿Qué es java?

Java es un lenguaje de programación de alto nivel y orientado a objetos. Con una sintaxis fácil de leer y escribir, se destaca por su portabilidad y capacidad de ejecutarse en diversas plataformas. Ampliamente utilizado en desarrollo de aplicaciones empresariales, Android, y sistemas embebidos.

3.Mysql

¿Qué es Mysql?

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto. Es ampliamente utilizado para almacenar y gestionar datos en aplicaciones web y empresariales. MySQL utiliza el lenguaje SQL para consultas y ofrece rendimiento, confiabilidad y escalabilidad.
